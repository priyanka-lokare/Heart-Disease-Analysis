# Heart Disease Analysis

## 📌 Project Overview
This project analyzes a healthcare dataset to identify key factors affecting heart disease.

## 🔧 Tools Used
- Python (Pandas)
- Excel
- Statistics

## 📊 Key Steps
- Data Cleaning (Handled missing values)
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Hypothesis Testing

## 📈 Key Insights
- Age and cholesterol levels strongly impact heart disease
- High blood pressure increases risk

## 🎯 Conclusion
This analysis helps in identifying high-risk patients and supports better decision-making.
